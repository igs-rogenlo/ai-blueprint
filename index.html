理解 AI Agent 的核心概念

A

Agent（代理人）

你的專屬 AI 助理。你給目標，它自己拆解步驟、選擇工具、完成任務。不只是聊天，而是主動執行。

S

Skill（技能）

教會 Agent 的固定工作流程。在 AGENTS.md 定義 SOP，Agent 記憶後就能重複執行。

例：「查營收產圖表」就是一個 Skill

M

MCP（即時連線協議）

Model Context Protocol — Agent 與外部工具之間的即時雙向通道。工具以 MCP Server 形式常駐運行，Agent 隨時呼叫。

例：MSSQL MCP Server、LanceDB 記憶系統

R

API（請求式呼叫）

傳統的 REST API — Agent 需要資料時發送請求、等待回應。不需常駐 Server，用完即走。

例：BigQuery API、Redmine API、QuickChart API

傳統工具 vs AI Agent

從被動回答到主動執行
